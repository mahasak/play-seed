Play SEED Project
=================================
This project based on Play 2.6 + Scala. Follow Reactive (http://www.reactivemanifesto.org/).


Prerequsit
===========
- Intelli J 2016
- JDK 8
- brew install -y sbt


Test Run
=====
- sbt run
- http://localhost:9000/system/status


Standard Command
===========
- sbt clean | compile | test | dist
